# Supervisor App
- приложение для отслеживания статусов задач.

## Стек
- React
- TypeScript
- React Router
- Zustand
- TanStack Query
- Axios
- Yup
- SCSS
- Vite

## Возможности
- Авторизация пользователя
- Просмотр списка задач
- Добавление задачи
- Редактирование задачи
- Удаление задачи
- Пагинация
- Кастомный Dropdown
- Модальные окна
- Валидация форм
